# Christmas Party Program

An interactive program flow for your Christmas Party.

## Features
- **Interactive Timeline**: Tap events to highlight them.
- **Christmas Aesthetics**: Snowfall, festive colors, and animations.
- **Mobile First**: Designed to be viewed on guests' phones.

## How to Run Locally
1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000

## How to Deploy to Vercel
1. Push this code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and sign in.
3. Click "Add New..." -> "Project".
4. Import your GitHub repository.
5. Click "Deploy".
